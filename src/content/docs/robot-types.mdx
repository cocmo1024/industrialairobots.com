Robot type pages will organize the major system classes used in industrial environments and explain where each one fits.

## Coverage priorities

- Articulated robots, cobots, gantries, and specialized platforms
- Strengths, limitations, and operating envelopes
- How robot class affects application fit and cell design

The purpose is to support real-world system selection, not abstract taxonomy.
