Applications are the core organizing layer for this site. Industrial robotics is best understood by the work being automated, not by hardware labels alone.

## Focus

- Material handling, inspection, tending, and assembly contexts
- Robotics use cases differentiated by process realities
- Application pages that connect robot classes, sensing, and deployment tradeoffs

This section is intended to become the main entry point for long-term readers.
