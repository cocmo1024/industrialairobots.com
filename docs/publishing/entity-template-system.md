# Industrial AI Robots Template System

This repository is designed for AI-operated publishing. New pages should follow the shared frontmatter contract and the matching outline for the page type.

## Shared frontmatter

Use these fields on new pages:

- `title`
- `description`
- `referenceType`
- `adProfile`
- `commercialIntent`
- `reviewCadence`
- `primaryKeyword`
- `searchIntent`
- `decisionStage`
- `targetRoles`
- `contentStatus`

## Application template

Use for job-focused pages like palletizing, machine tending, inspection, and picking.

Sections:

1. Why the application matters
2. Process variability and throughput pressures
3. Robot or sensing implications
4. Cell-design implications
5. Compare next

## Case study template

Use for pilot, rollout, and implementation-pattern pages.

Sections:

1. What the project pattern is
2. What the pilot or rollout needed to prove
3. Hidden failure points
4. Why the outcome matters
5. Related deployment pages

## Robot type template

Use for cobots, articulated robots, SCARA, gantry, and similar categories.

Sections:

1. Where the robot class fits
2. Where it is often overused
3. Operational tradeoffs
4. Deployment consequences
5. Compare next

## Deployment template

Use for ROI, pilot design, readiness, maintenance, and change-management pages.

Sections:

1. Decision objective
2. What should be validated early
3. Why ROI models fail
4. Organizational readiness
5. Next-step pages
